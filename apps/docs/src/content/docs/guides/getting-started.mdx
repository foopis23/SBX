### Spawn a Cube

```diff
		import {
			Game,
			Graphics,
			// Physics,
			// Audio,
+			Transform
		} from '@repo/sbx';

		const game = new Game();
		Graphics.initDefault(game);
		// Physics.initDefault(game);
		// Audio.initDefault(game);

-		game.start()
+		game.start(() => {
+			game.world.spawn(
+				Transform(),
+				Graphics.Mesh(
+					new Graphics.Three.Mesh(
+						new Graphics.Three.BoxGeometry(1, 1, 1),
+						new Graphics.Three.MeshBasicMaterial({
+							color: 0xFF0000
+						})
+					)
+				)
+			)
+		})
```

### Add a System to Rotate All Objects

```diff
		import {
			Game,
			Graphics,
			// Physics,
			// Audio,
			Transform
		} from '@repo/sbx';

		const game = new Game();
		Graphics.initDefault(game);
		// Physics.initDefault(game);
		// Audio.initDefault(game);

+		// spin all objects system
+		game.update.add(({game, delta}) => {
+			game.world.query(Transform).updateEach(([transform]) => {
+				transform.rotation.x += delta * 2
+				transform.rotation.y += delta * 2
+				transform.rotation.z += delta * 2
+			})
+		})

		game.start(() => {
			game.world.spawn(
				Transform(),
				Graphics.Mesh(
					new Graphics.Three.Mesh(
						new Graphics.Three.BoxGeometry(1, 1, 1),
						new Graphics.Three.MeshBasicMaterial({
							color: 0xFF0000
						})
					)
				)
			)
		})
```

## Further reading

- Read [about koota](https://github.com/pmndrs/koota). Koota is the ECS framework SBX uses.
- Read [about directed]('https://github.com/pmndrs/directed'). Directed is the Scheduler SBX uses for system.
- Read [about threejs]('https://threejs.org/'). ThreeJS is the graphics engine SBX uses.
- Read [about rapier]('https://rapier.rs/docs/user_guides/javascript/getting_started_js'). Rapier is the Physics Engine SBX uses.
